Files in src:
--------------
Main.java.              Main class
URLInfo.java.           Stores details of the URL
GetContentLength.java.  Gets length of a file in bytes
GetURLImage.java.       Reads image URL
GetURLText.java.        Reads html or text URL
GetURLDocument.java.    Reads doc or pdf URL


Compile & Run Code:
-------------------
Type input url in input_urls.txt file then
javac -d . *.java
java -cp . Main  -i input_urls.txt -o output.txt -d data
In the Data folder, the URL info would be downloaded
